How to get a token:
1. Go to github.com/settings/tokens/new
2. Give it a name (e.g. "Portfolio Admin")
3. Set expiration to your preference
4. Under Scopes, check repo (full control of private repos)
5. Click Generate token and paste it above

Your token is saved only in your browser's localStorage — it is never sent anywhere except directly to the GitHub API.
